# Port Legacy Migration Reference

This directory is retained from the initial Port-based IDP experiment. It remains useful as a migration reference for the original blueprint, entity, scorecard, action, and automation concepts.

Backstage is now the target platform for the self-hosted MVP. New target artifacts should be added under `backstage/`, `examples/services/*/catalog-info.yaml`, `.github/ISSUE_TEMPLATE/`, and the wiki docs.

Do not add new Port-only capabilities unless a task explicitly asks for migration comparison or legacy cleanup.
